# ✍️ Handwriting Simulator for macOS

A native macOS app that simulates realistic handwriting on lined, graph, or blank paper.
Perfect for creating homework videos for **Class 11 Mago's** (Physics, Chemistry, Maths, etc.).

## Features
- Write any text using a beautiful cursive font (Apple Chancery)
- Realistic ink simulation with variable pressure and bleeding
- Three paper types: lined (with red margin), graph, blank textured
- Adjust writing speed
- Animated drawing on screen
- Export as a high‑quality **QuickTime movie** (1920×1080)

## How to Run
1. Open `HandwritingSimulator.xcodeproj` in Xcode 14+.
2. Select the target **HandwritingSimulator > My Mac**.
3. Build and run (⌘R).

## Usage
- Type your homework text in the left panel.
- Select paper, ink color, and speed.
- Press **Animate Writing** to watch it draw.
- Press **Export Movie** to save a video file.# Mac
